      Formative Task 7

      1. Carefully go through Scheduling and Patterns by reading the material and evaluating/studying the code.
      2. Complete 8 Synthesis Definitions (SynthDef) using each of following synthesis techniques:
        1. Sample-based synthesis
        2. Substrctive synthesis
        3. Additive synthesis (summation)
        4. Wavetable-lookup
        5. Wave shaping
        6. Wavetable crossfading and/or wave stacking
        7. Modulation synthesis (Simple FM)
        8. A synth using resonance (using Formlet, Klank, and/or CombL)

        NOTE: you can work in groups for this

      3. Recommended Reading: pp. 267-304 (Ch 17-20) from Designing Sound (Farnell 2010)
