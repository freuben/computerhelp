            Description

            A second order low pass filter.

            Class Methods

            *ar (in = 0, freq = 440, mul = 1, add = 0)

            *kr (in = 0, freq = 440, mul = 1, add = 0)

            Arguments:

            in

            The input signal.

            freq

            Cutoff frequency in Hertz.

            mul

            Output will be multiplied by this value.

            add

            This value will be added to the output.
