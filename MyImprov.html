        Description

        MyImprov is a modular command-style environment for improvisation. Short command-style code is evaluated in a call window from which you can access different instruments, triggers and filters. The synthax is described by accessing the help - typing "help" in the call window. MyImprov uses JITLib but simplifies its code and interpretation.
